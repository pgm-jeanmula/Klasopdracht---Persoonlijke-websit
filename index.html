Projecten

Eindopdracht Mockify Programming 2

Mockify is een eenvoudige RESTful API die geïnspireerd is op bepaalde onderdelen van de Spotify API. Deze applicatie werd ontwikkeld als eindopdracht voor het vak Programming 2 (PGM 2). De API laat toe om tracks (nummers) en playlists (afspeellijsten) te beheren via HTTP-requests. De focus ligt op correcte REST-conventies, inputvalidatie, error handling, testing en documentatie.

Voor deze opdracht werk je een responsive web applicatie uit voor de Tijdloze (100). De Tijdloze 100 is de "allertijdenmuzieklijst" van Studio BrusselKoppelingen naar een externe site..
